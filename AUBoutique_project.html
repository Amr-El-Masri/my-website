AUBoutique: A Marketplace for the AUB Community

AUBoutique is a full-stack marketplace application designed for the American University of Beirut (AUB) community. The platform enables students and members to buy, sell, and exchange items within a trusted university network, encouraging sustainable consumption and reducing unnecessary waste.

The system is built using a hybrid client–server and peer-to-peer architecture. Core services such as authentication, product management, messaging, and database operations are handled by a multithreaded TCP server, while optional peer-to-peer communication allows users to chat directly without server relay.

  • Secure user authentication with password hashing (bcrypt)
  • Concurrent server and client handling using multithreading
  • Marketplace with product listings, search, filtering, and ratings
  • Direct peer-to-peer and server-based messaging

System Architecture

AUBoutique follows a hybrid communication model. Each client connects to a central server responsible for authentication, database access, and coordination. The server maintains active connections using TCP sockets and creates a dedicated thread for every connected user.

For real-time communication, users may either exchange messages through the server or initiate a peer-to-peer connection by retrieving the recipient’s IP and port from the server. This design balances reliability with scalability and reduces server load during active chats.

Key Features

  • User registration, login, and authentication
  • Product selling, purchasing, and inventory management
  • Image upload and download using chunked TCP transfers
  • Product ratings and average rating computation
  • Search, filtering by owner, and partial currency conversion
  • Creative budgeting feature to help students manage spending

Technologies Used

  • Python (Sockets, Multithreading)
  • TCP/IP Networking
  • SQLite for database management
  • Bcrypt for secure password hashing
  • Client-server & peer-to-peer communication models
